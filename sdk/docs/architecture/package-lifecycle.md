# Package Lifecycle

## Author
Write `.lithic` source files and package metadata.

## Build
Compile into bytecode + ABI + manifest + storage and event schemas.

## Test
Run unit, property, conformance, and simulation tests.

## Deploy
Deploy to Makalu or a compatible target profile using generated artifacts.

## Operate
Track emitted events, fulfill receipts, verify proofs, and audit upgrades.

## Upgrade
Run storage compatibility checks, manifest version bumps, and governance flow.
